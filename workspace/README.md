# Workspace

`workspace/` is Francis's live local runtime substrate.

It is not source code and it is not meant to be a durable git history of:

* receipts
* ledgers
* inbox messages
* incidents
* telemetry streams
* queue state
* takeover exports
* Forge staging packs
* Forge reports
* temporary chain and tool runtime artifacts

Those files are generated by Francis while the system runs. They are allowed to change constantly and should stay local to the machine unless they are intentionally exported or promoted elsewhere.

Git tracks only this policy file and the local ignore rules in [`.gitignore`](./.gitignore).

If a runtime artifact needs to become a product asset, test fixture, or canonical example, promote it into a real repo surface such as:

* `docs/`
* `tests/fixtures/`
* `packages/`
* `services/`

Do not recommit raw `workspace/` churn by default.
